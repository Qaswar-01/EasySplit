# EasySplit

EasySplit is a modern web application designed to simplify expense splitting and group financial management. Built with React and Vite, it offers an intuitive interface for managing shared expenses, group budgets, and financial tracking.

## Features

- **Group Management**: Create and manage expense groups with multiple members
- **Expense Tracking**: Add and track expenses with detailed categorization
- **Quick Actions**: Easily add expenses and settle balances
- **Dashboard**: View balances and financial summaries at a glance
- **Dark Mode**: Toggle between light and dark themes for comfortable viewing
- **Mobile Responsive**: Fully responsive design that works on all devices

## Tech Stack

- React
- Vite
- Tailwind CSS
- Local Storage for data persistence

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Qaswar-01/EasySplit.git
   ```

2. Navigate to the client directory
   ```bash
   cd EasySplit/client
   ```

3. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

4. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. Open your browser and visit `http://localhost:5173`
